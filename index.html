Comandos Git Bash

Comando Git Exemplo de uso Operação
config git config --global user.mame | user.email configura o nome do usuário e o email
help git help < comando > obtém ajuda sobre um comando expecífico
init git init inicializa um repositório em uma pasta
status git status verifica a situação do repositório
add <file> git add . | git add <arquivo> adiciona um ou mais arquivos ao staging area
commit git commit -M "mensagem" commita o repositório
reset git reset <arquivo> volta do estado preparado para o modificado
restore git restore <arquivo> desfaz uma alteração em um arquivo modificado
clean git clean -fd apaga um arquivo criado
branch git branch -M 'main' mudar o nome da branch principal
code code . | code <arquivo< abre o vsCode na pasta ou arquivo
